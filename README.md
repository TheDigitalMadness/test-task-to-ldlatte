# Запуск
- Перейти в корень проекта
- `docker compose up -d --build` - запустить в фоновом режиме
- Приложение доступно по адресу `http://localhost:8080`
- Swagger доступен по адресу `http://localhost:8080/api/swagger`

# Функциональность сервиса
- Все защищенные эндпоинты доступны с JWT. При отсутствии авторизации, юзера перекидывает на страницу авторизации
- На главной странице присутствует несколько разделов:
- - Посередине - встречи на выбранный период
- - Выше - переключатели день-неделя, поле выбора даты, встречи на которую интересно посмотреть, фильтр "только мои" и кнопка создания встречи
- - Справа - окно со всеми встречами юзера
- Создание встречи:
- 1. По кнопке "Создать встречу" открывается модалка
- 2. Вводится тема и выбирается до 15 участников встречи.
- 3. Далее выбирается дата встречи
- 4. По нажатию "Далее", если встреча успешно создана, она добавиться в БД и появится в списке
- 5. Если же кто-то из участников на эту дату не свободен, юзера перебросит на страницу выбора участников, где красным подсветит занятых участников

# Структура
- `/backend` - python - FastAPI backend
- `/frontend` - React frontend (сделан с ИИ)
- `/gateway` - внешний nginx

# Бэкенд стэк
- FastAPI - framework
- PostgreSQL - DB
- SQLAlchemy - ORM
- pydantic - validation
