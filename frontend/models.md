# Модели фронтенда

## User

Пользователь системы, которого можно выбрать участником встречи.

| Поле | Тип | Описание |
| --- | --- | --- |
| `id` | `number` | Уникальный идентификатор пользователя |
| `username` | `string` | Отображаемый юзернейм, уникальный |
| `login` | `string` | Логин для авторизации, уникальный |
| `fullName` | `string` | Имя сотрудника для интерфейса |

## AuthUser

Текущий авторизованный пользователь.

| Поле | Тип | Описание |
| --- | --- | --- |
| `id` | `number` | Идентификатор пользователя |
| `username` | `string` | Юзернейм |
| `login` | `string` | Логин |
| `fullName` | `string` | Имя сотрудника |
| `token` | `string` | JWT, который фронтенд хранит в `localStorage` и передаёт в `Authorization: Bearer <token>` |

## SignInPayload

Тело запроса на вход существующего пользователя.

| Поле | Тип | Описание |
| --- | --- | --- |
| `login` | `string` | Логин пользователя |
| `password` | `string` | Пароль пользователя |

## SignUpPayload

Тело запроса на регистрацию нового пользователя.

| Поле | Тип | Описание |
| --- | --- | --- |
| `login` | `string` | Логин пользователя, уникальный |
| `password` | `string` | Пароль пользователя |
| `username` | `string` | Юзернейм пользователя, уникальный |
| `fullName` | `string` | Фамилия и имя пользователя, собранные фронтом из отдельных полей |

## Meet

Встреча в календаре. В текущем ТЗ дата выбирается отдельными полями день/месяц/год, поэтому слот считается занятым на выбранную дату.

| Поле | Тип | Описание |
| --- | --- | --- |
| `id` | `UUID` | Уникальный идентификатор встречи |
| `topic` | `string` | Тема встречи |
| `date` | `string` | Дата встречи в формате `YYYY-MM-DD`; только будущая дата при создании |
| `meet_users` | `MeetUser[]` | Связки встречи с пользователями |
| `createdBy` | `number` | ID пользователя, создавшего встречу |
| `createdAt` | `string` | ISO-строка даты создания |

## MeetUser

Связка встречи с пользователем.

| Поле | Тип | Описание |
| --- | --- | --- |
| `id` | `UUID` | Уникальный идентификатор связки |
| `meet_id` | `UUID` | ID встречи |
| `user_id` | `number` | ID пользователя |

## CreateMeetPayload

Тело запроса на создание встречи.

| Поле | Тип | Описание |
| --- | --- | --- |
| `topic` | `string` | Тема встречи |
| `participantIds` | `number[]` | ID выбранных участников |
| `date` | `string` | Дата встречи в формате `YYYY-MM-DD` |

## ApiError

Единый формат ошибки для фронтенда.

| Поле | Тип | Описание |
| --- | --- | --- |
| `status` | `number` | HTTP-статус |
| `message` | `string` | Сообщение об ошибке |
| `conflicts` | `UUID[] \| undefined` | Только для `409`: ID связок `MeetUser`, у которых уже есть встреча на выбранную дату |
